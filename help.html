Requirements
Web surfer will run on almost any PC with a Pentium or equivalent processor. Web surfer uses files from Internet Explorer, so you will need IE installed. The screen resolution must be at least 800 X 600. A small amount of disk space is used (less than 2 megabytes).
 
 
 
 
 
 

Setting up the Internet connection
You need to have Internet Explorer installed on the PC in order to use Web Surfer. The Internet settings are accessed from Internet Explorer Option menu. The settings are provided by your ISP.
 
 
 
 
 
 

Using Ad control
The Advertisement control setup is described in the Toolbar Option menu. This feature is usefull in controlling those annoying pop-up ads. You can customize the Ad control settings to reduce aggravation while surfing the net. Web surfer can automatically close those ad windows for you.
 
 
 
 
 

Printing and saving documents and pictures
Right clicking on a page brings up a menu which gives you several options. The options available depend on where you click. If you wish to view, print, or save the source code then click on a blank area of the web page. If you click on a picture, you can save or print that picture. Other options are available.
 
 
 
 

Setting up Web surfer as the default browser
You can set up Windows so it will launch Web surfer when you open an html file. Follow this procedure: Open Windows Explorer. Click View, Folder Options..., and File Types. Scroll down to HTML Document and click on it. Click Edit..., Open, and Edit... In the box labeled "Application used to perform action:" enter C:\websur~1\websur~1.exe "%1". If Use DDE is checked, uncheck it. Click OK, OK, and OK.
Now when you open an html file, it will launch Web surfer. If you are offline, a window may appear asking you to log on. If you want to work offline, just click cancel. The page will load.
