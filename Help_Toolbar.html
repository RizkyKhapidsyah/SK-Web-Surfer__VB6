Toolbar

Home
Loads the home page. The home page can be changed using the options button.
 
 
 
 

Refresh
Refreshes the screen by reloading the current web page.
 
 
 
 

Stop
Aborts a web site download.
 
 
 
 

Catalog
Displays the Catalog window. The Catalog is used to store and recall web site addresses. Both the URL and site name are stored. To store the current website in the catalog, click the 'Add the Current Site to the Catalog' button. Then press OK to return to Web Surfer. To recall a web site from the catalog, Click on the web site URL or name in the book. An open folder icon will appear to the left of the site name. Then click OK. There are up to 13 pages in the catalog. To change pages, use the slider bar at the bottom of the Catalog window. You can also search the catalog for a keyword. Type the keyword into the text box to the right of 'Select from the Catalog, or search for:'. Press Enter or click on the binoculars. The catalog will be searched for the keyword. To delete a web site from the Catalog, first click on the site name or URL. An open folder icon will appear to the left of it. Then click the 'Delete from Catalog' button. Click OK or Cancel to return to Web Surfer.
 
 
 
 

Search
Loads the internet search website. The search website can be changed using the options button.
 
 
 
 

Back
Re-loads the previous website. To go several steps back, hold the left mouse button down for a moment. A selection box will appear. Slide the mouse over the selection box. Select a web site and click on it.
 
 
 
 

Forward
Re-loads the website from before the back button was clicked.
 
 
 
 

Sports
Loads the Internet sports site. The sports website can be changed using the options menu.
 
 
 
 

Weather
Loads the Internet weather site. The weather website can be changed using the options menu.
 
 
 
 

News
Loads the Internet news website. The news website can be changed using the options menu.
 
 
 
 

About
Displays the software version. Also accesses the help menu.
 
 
 
 

Options
Displays the options menu.The current URL's for home page, sports, weather, news, and search sites are shown. To set the home page to the current site, click on 'Set the Home Page to the Current Page'. The red box will change to a red dot. Then click OK. To set the home page to a different site, type the URL into the box to the right of 'Set the Home Page to:'. Click on 'Set the Home Page to:'. The red box will change to a dot. Click OK.
Clicking on 'Clear History' then OK will erase the recent site history.
To change the sports, weather, news, or search sites use the same procedure described above for 'Set the Home Page to:'. (By right clicking the mouse on a text box you can cut, copy, or paste.)
Under Ad Control, you have three options. When 'Exterminate all New Windows' is chosen, all Explorer or Netscape windows spawned by a website will be closed automatically. When 'Ask' is chosen, a dialog box will appear when a new window is spawned. Some sites may generate numerous pop-up ads. You can select which windows you want closed and which you wish to keep open. If you choose to keep a window open, you will not be asked about that window again. If 'Allow new Windows' is chosen then all windows spawned will be left opened. You will have to close them yourself.
The 'Load Toolbar Theme' option changes the background and icons in the toolbar. Clicking on it opens a file select box. Select a toolbar theme (.TBT) file. You can also create your own toolbar themes.
 
 
 
 

Load File
Loads a file from a disk. A file selection box is displayed. Browse the computer for an html file to display on the Web Surfer.
 
 
 
 

Full Screen
Hides the Toolbar and Address bar. Click on the icon in the upper right corner to show the Toolbar and Address bar.
 
 
 
 

Toolbar location
Allows the toolbar to be positioned at the top, bottom, left, or right side of the window.
 
 
 
 

Exit
Exits Web Surfer. You should use this button rather than the X in the upper right corner.
